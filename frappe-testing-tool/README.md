# Frappe Framework Testing Tool

## Overview
A comprehensive Single Page Application (SPA) for testing Frappe Framework projects with advanced testing capabilities.

## Features
- GitHub Repository Analysis
- Multiple Testing Types Support
  - Unit Testing
  - Integration Testing
  - System Testing
  - Functional Testing
  - Performance Testing
  - Security Testing
  - User Acceptance Testing
  - Regression Testing
  - Mobile & Responsive Testing
  - Compatibility Testing

## Prerequisites
- Node.js (v14 or later)
- npm or Yarn
- Git
- GitHub Repository with Frappe Framework Project

## Installation
1. Clone the repository
```bash
git clone https://github.com/yourusername/frappe-testing-tool.git
cd frappe-testing-tool
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm start
# or 
yarn start
```

## Usage
1. Enter GitHub Repository URL
2. Analyze Repository
3. Select Testing Types
4. Configure Test Environment
5. Run Tests
